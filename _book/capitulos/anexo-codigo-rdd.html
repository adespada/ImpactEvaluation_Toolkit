32  Código R: Regresión discontinua

NotaSobre este anexo

Este anexo contiene el código R completo para el taller práctico de Regresión Discontinua (RDD). Para la teoría y explicación del método, consulte el capítulo correspondiente.

32.1 Taller práctico

NotaSobre este taller

Vamos a trabajar con datos simulados de un programa de subsidio de vivienda. La elegibilidad se determina por un puntaje de pobreza: los hogares con puntaje menor a 50 reciben el subsidio. El resultado es un índice de calidad de vivienda. Tenemos 500 hogares.

32.1.1 Paso 1: Crear los datos simulados

library(rdrobust)

set.seed(2024)
n <- 500

puntaje <- runif(n, min = 15, max = 85)
corte <- 50
subsidio <- ifelse(puntaje < corte, 1, 0)

efecto_subsidio <- 8
calidad <- 30 + 0.4 * puntaje + efecto_subsidio * subsidio + rnorm(n, sd = 5)

datos_rdd <- data.frame(
  id = sprintf("HOG_%03d", 1:n),
  puntaje = round(puntaje, 1),
  subsidio = subsidio,
  calidad = round(calidad, 1)
)

head(datos_rdd, 10)

32.1.2 Paso 2: Visualizar la discontinuidad

library(ggplot2)

datos_rdd$grupo <- ifelse(datos_rdd$subsidio == 1, "Con subsidio", "Sin subsidio")

ggplot(datos_rdd, aes(x = puntaje, y = calidad, color = grupo)) +
  geom_point(alpha = 0.4, size = 1.5) +
  geom_smooth(method = "lm", se = TRUE, linewidth = 1.2) +
  geom_vline(xintercept = corte, linetype = "dashed",
             color = "#BA7517", linewidth = 0.9) +
  scale_color_manual(values = c("Con subsidio" = "#D85A30",
                                 "Sin subsidio" = "#185FA5")) +
  labs(x = "Puntaje de pobreza", y = "Índice de calidad de vivienda",
       color = NULL,
       title = "Regresión discontinua: efecto del subsidio de vivienda") +
  theme_minimal(base_size = 13) +
  theme(legend.position = "bottom",
        plot.title = element_text(face = "bold"))

32.1.5 Paso 5: Robustez con diferentes anchos de banda

bw_optimo <- modelo_rdd$bws[1, 1]

bw_mitad <- rdrobust(datos_rdd$calidad, datos_rdd$puntaje, c = corte,
                      h = bw_optimo / 2)
bw_doble <- rdrobust(datos_rdd$calidad, datos_rdd$puntaje, c = corte,
                      h = bw_optimo * 2)

robustez <- data.frame(
  Ancho_de_banda = c(
    paste0("Mitad (", round(bw_optimo / 2, 1), ")"),
    paste0("Óptimo (", round(bw_optimo, 1), ")"),
    paste0("Doble (", round(bw_optimo * 2, 1), ")")
  ),
  Efecto = c(round(bw_mitad$coef[1], 2), round(modelo_rdd$coef[1], 2), round(bw_doble$coef[1], 2)),
  Error_Estandar = c(round(bw_mitad$se[1], 2), round(modelo_rdd$se[1], 2), round(bw_doble$se[1], 2)),
  Valor_p = c(round(bw_mitad$pv[1], 3), round(modelo_rdd$pv[1], 3), round(bw_doble$pv[1], 3))
)
