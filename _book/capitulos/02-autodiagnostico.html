8  Herramienta de autodiagnóstico

  • Mostrar todo el código
  • Ocultar todo el código
  • Ver el código fuente

Esta herramienta interactiva le ayuda a identificar qué método(s) de evaluación de impacto son más apropiados para su programa. Está basada en la herramienta Choosing Appropriate Evaluation Methods desarrollada por Barbara Befani y Michael O’Donnell para BOND UK y actualizada por Barbara Befani para CECAN.

ImportanteAdvertencia importante

Esta herramienta es una ayuda para la toma de decisiones, no un sustituto del juicio profesional. Los resultados deben discutirse con evaluadores expertos antes de tomar una decisión final. Considere los resultados como un punto de partida para la conversación, no como una receta definitiva.

  const arr = [];

    let ess_unmet = 0;

      if (req > 0) {

        if (ans < 0) {
          if (req >= 1) ess_unknown++;
        } else if (ans === 0) {
          if (req >= 1) ess_unmet++;
        } else {
          fscore += req * ans;
        }
      }

    const sc = fmax > 0 ? Math.round((fscore / fmax) * 100) : 100;

      name: methodNames[i], type: methodTypes[i], chapter: methodChapters[i],
      score1: s2_scores[i].score1, score2: s2_scores[i].score2, score3: sc,
      ess_unmet: ess_unmet, ess_unknown: ess_unknown, idx: i

8.2 ¿Cómo interpretar estos resultados?

TipGuía de interpretación
  • Puntaje combinado > 70%: El método es una buena opción para su contexto. Consulte el capítulo correspondiente.
  • Puntaje combinado 50-70%: El método podría funcionar pero tiene limitaciones. Evalúe si puede resolver las restricciones.
  • Puntaje combinado < 50%: El método probablemente no es adecuado para su situación actual.
  • Barras en rojo: El método tiene requisitos esenciales que usted indicó que no puede cumplir. Considere si esa restricción es resoluble.
  • Si ningún método supera el 70%: Considere fortalecer su sistema de monitoreo (vuelva a la Parte 1) o explore diseños mixtos (Parte 5).
NotaSobre la combinación de métodos

Es raro que un solo método responda a todas las preguntas de evaluación. Si sus preguntas combinan interés en el efecto neto (Etapa 1, pregunta 1.1) con interés en los mecanismos causales (pregunta 1.3), probablemente necesitará combinar un método cuantitativo con uno cualitativo.

        } else {

## ¿Cómo interpretar estos resultados?

::: {.callout-tip}
## Guía de interpretación

- **Puntaje combinado > 70%:** El método es una buena opción para su contexto. Consulte el capítulo correspondiente.
- **Puntaje combinado 50-70%:** El método podría funcionar pero tiene limitaciones. Evalúe si puede resolver las restricciones.
- **Puntaje combinado < 50%:** El método probablemente no es adecuado para su situación actual.
- **Barras en rojo:** El método tiene requisitos esenciales que usted indicó que no puede cumplir. Considere si esa restricción es resoluble.
- **Si ningún método supera el 70%:** Considere fortalecer su sistema de monitoreo (vuelva a la [Parte 1](01-marco-logico.qmd)) o explore diseños mixtos ([Parte 5](05-por-que-combinar.qmd)).
:::

::: {.callout-note}
## Sobre la combinación de métodos
Es raro que un solo método responda a todas las preguntas de evaluación. Si sus preguntas combinan interés en el efecto neto (Etapa 1, pregunta 1.1) con interés en los mecanismos causales (pregunta 1.3), probablemente necesitará combinar un método cuantitativo con uno cualitativo.
:::
