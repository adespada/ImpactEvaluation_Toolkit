31.1 Taller práctico

NotaSobre este taller

Vamos a trabajar con datos simulados de un programa de becas universitarias. El gobierno otorgó becas mediante una lotería entre postulantes elegibles. Queremos estimar el efecto de asistir a la universidad (tratamiento) sobre el ingreso laboral (resultado), usando la lotería (instrumento) para corregir el sesgo de selección. Tenemos 500 individuos.

31.1.1 Paso 1: Crear los datos simulados

library(AER)

set.seed(2024)
n <- 500

habilidad <- rnorm(n, mean = 0, sd = 1)
loteria <- rbinom(n, 1, prob = 0.5)

prob_universidad <- plogis(-0.5 + 1.5 * loteria + 0.8 * habilidad)
universidad <- rbinom(n, 1, prob = prob_universidad)

efecto_universidad <- 350
ingreso <- 800 + efecto_universidad * universidad + 150 * habilidad + rnorm(n, sd = 200)

datos_iv <- data.frame(
  id = sprintf("ID_%03d", 1:n),
  loteria = loteria,
  universidad = universidad,
  ingreso = round(ingreso)
)

head(datos_iv, 10)

Advertencia¿Por qué OLS está sesgado?

El coeficiente OLS sobreestima el efecto de la universidad porque las personas con mayor habilidad innata tienen más probabilidad de ir a la universidad y de tener mayores ingresos.

31.1.5 Paso 5: Comparar OLS vs IV

comparacion <- data.frame(
  Metodo = c("OLS (sesgado)", "IV / 2SLS (corregido)", "Efecto real (simulado)"),
  Coeficiente = c(
    round(coef(modelo_ols)["universidad"], 1),
    round(coef(modelo_iv)["universidad"], 1),
    350
  ),
  Error_Estandar = c(
    round(summary(modelo_ols)$coefficients["universidad", "Std. Error"], 1),
    round(summary(modelo_iv)$coefficients["universidad", "Std. Error"], 1),
    NA
  )
)

knitr::kable(comparacion,
             col.names = c("Método", "Efecto estimado", "Error estándar"))

library(ggplot2)

resultados <- data.frame(
  metodo = c("OLS\n(sesgado)", "IV / 2SLS\n(corregido)"),

ggplot(resultados, aes(x = metodo, y = estimado)) +
